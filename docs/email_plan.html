Phase 1 - Foundation + Manual Mode

Deliverable: Paste a job, get a proposal on screen

  1. Scaffold MVC framework (core/, config/, public/index.php, .htaccess, .env)
  2. Create freelance MySQL database + run all migrations
  3. Auth system - login, logout, register, forgot/reset password, session guards
  4. Admin dashboard - left sidebar nav, stats overview (jobs, proposals, win rate)
  5. Resume management - upload PDF/DOCX, extract plain text, CRUD, set active
  6. Platform management - seed 5 platforms, enable/disable, configure parser class
  7. Proposal rules - CRUD by category, drag-and-drop reorder, seed defaults (honest about skills, 26yr exp, availability Apr 1 2026, rate suggestions, flag weak fits, match tone to client)
  8. Manual job entry - paste job posting, select platform, parse into structured fields
  9. Claude API integration - ProposalGenerator: resume + job + rules + availability = proposal + fit score + rate
  10. Proposal view/edit - display, edit, copy-to-clipboard, regenerate with different tone
  11. Apache vhost - freelance.local
  12. Git init + .gitignore + first commit

Phase 2 - Platform Parsers + Job History

Deliverable: Per-platform parsing, job/proposal tracking, calendar

  1. Platform-specific parsers - Upwork, Wellfound, Contra, Turing, Freelancer
  2. Job history and search - paginated, filter by platform/status/fit score
  3. Proposal history - versions, status tracking (drafted/submitted/won/lost)
  4. Availability calendar - visual calendar, scheduling windows, hours per week
  5. Dashboard analytics - proposals per platform, win rate, API usage/cost

Phase 3 - Automated Email Pipeline

Deliverable: Job alert to proposal in your inbox (hands-free)

  1. Pipe-to-script setup on AWS - incoming mail to jobs@visionquest2020.net triggers PHP parser instantly (IMAP poll fallback)
  2. Gmail forwarding - filters per platform alert emails forward to jobs@visionquest2020.net
  3. EmailPipeline.php - identify platform, route to parser, create Job, check fit score (below 5 = skip + notify, 5+ = generate proposal), email draft via claude_messenger
  4. Register in claude_mailbox poller for phone-based approve/edit/reject

Phase 4 - Polish + Production

Deliverable: Live at freelance.visionquest2020.net

  1. PDF export of proposals (DOMPDF)
  2. Security hardening - CSRF, CSP, input sanitization, rate limiting
  3. PHPUnit + Playwright tests
  4. Deploy script (git-pull to AWS 50.16.139.240)
  5. Production .env - RDS credentials, API key, mail config
  6. SSL - Lets Encrypt for freelance.visionquest2020.net
  7. Nightly automation - .claude-nightly config
  8. Cost estimate baseline

Core Engine

ProposalGenerator builds Claude API call from: system prompt + rules, active resume, parsed job, availability windows, platform context (adjusts tone per platform).

Output: proposal_text, fit_score (1-10), fit_notes, suggested_rate, tone, skip_reason
